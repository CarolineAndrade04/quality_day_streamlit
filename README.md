# 📊 Quality Day Streamlit

Dashboard interativo desenvolvido em Python e Streamlit para demonstrar, de forma prática, como dados podem ser transformados em informação, conhecimento e suporte à tomada de decisão.

## 🎯 Objetivo

Este projeto foi criado para uma atividade interativa realizada durante uma semana temática de qualidade e inovação em ambiente industrial.

A proposta consistiu em coletar respostas dos participantes em tempo real e transformar essas informações em indicadores visuais, correlações e insights instantâneos, permitindo demonstrar conceitos de Ciência de Dados de forma acessível e aplicada.

---

## 🚀 Principais Funcionalidades

* Atualização automática dos dados em tempo real
* Dashboard interativo desenvolvido em Streamlit
* Indicadores executivos (KPIs)
* Correlações entre variáveis
* Visualizações interativas com Plotly
* Insights automáticos baseados nas respostas coletadas
* Interface customizada com HTML e CSS

---

## 📈 O que foi demonstrado

Através de uma dinâmica simples de coleta de dados, o projeto permitiu explorar conceitos como:

* Transformação de dados em informação
* Visualização de dados
* Estatística aplicada
* Correlação entre variáveis
* Construção de indicadores
* Apoio à tomada de decisão baseada em dados

## 🏭 Contexto de Negócio

Mais do que desenvolver um dashboard, o objetivo deste projeto foi aproximar conceitos de Ciência de Dados da realidade operacional.

A iniciativa foi utilizada como recurso de apoio em uma apresentação sobre transformação digital e análise de dados, demonstrando como informações simples, quando analisadas de forma estruturada, podem gerar novos insights e perspectivas sobre um problema.

## ▶️ Execução Local

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute a aplicação:

```bash
streamlit run app.py
```

A fonte de dados utilizada durante o evento foi removida da versão pública por motivos de privacidade e confidencialidade.
